# uncensored-ai-image-video-generator

An easy-to-use AI-powered image and video generation tool designed for Windows.  
`uncensored-ai-image-video-generator` provides a simple setup flow, local execution, and a clean workflow for generating AI images and videos from text prompts, reference images, or existing visual assets.

`uncensored-ai-image-video-generator` supports text-to-image generation, text-to-video generation, and image-to-video workflows, allowing users to create animated videos based on a source image. It also includes advanced image transformation tools for editing, modifying, and restyling images according to user prompts.

## Features

- Text-to-image generation
- Text-to-video generation
- Image-to-video generation based on a reference image
- Advanced AI image editing and transformation
- Prompt-based image modification
- Creative image restyling and editing
- Windows-friendly installation
- One-command setup
- Simple local workflow
- Custom prompt support
- Extendable project structure
